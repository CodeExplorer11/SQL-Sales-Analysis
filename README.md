# 📊 SQL Sales Analysis Project

A SQL-based business analytics project designed to analyze sales performance, customer behavior, product trends, and regional business insights using MySQL.

---

## ✨ Features

- Customer sales analysis
- Product performance tracking
- Regional revenue analysis
- Top-selling product identification
- Revenue and order analytics
- Business KPI generation

---

## 🛠️ Tech Stack

- MySQL
- MySQL Workbench
- SQL

---

## 📂 Database Tables

The project contains the following tables:

- Customers
- Products
- Orders
- Sales

---

## 🔥 SQL Concepts Used

- JOIN
- GROUP BY
- ORDER BY
- SUM()
- AVG()
- COUNT()
- HAVING

---

## 📊 Key Analyses Performed

- Top-selling products
- Best customers by revenue
- Regional sales analysis
- Monthly revenue trends
- Average order value
- Total sales revenue

---

## 🚀 How to Run

1. Open MySQL Workbench
2. Run the `sales_analysis.sql` file
3. Execute the queries to analyze sales data
4. View query outputs and business insights
